## 入选标准

- 已有稳定静态页面，可以由 GitHub Pages 直接访问。
- 作品表达一个独立想法，不只是后续作品已经吸收的阶段稿。
- 页面不依赖内部路径、服务返修中间件或未公开工作台文件。
- 涉及原典、手稿、研究图表或论断时，必须在页面内标清来源状态和证据边界。
- 研究试读可以公开展示问题意识和证据结构，但不能冒充最终论文定稿。

全部静态页面可直接由 GitHub Pages 托管。
